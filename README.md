# dynemsim
Error Mitigation in Dynamic Circuits for Hamiltonian Simulation

This repository contains code that was used to conduct experiments
that benchmark two standard error mitigation strategies :
Dynamical Decoupling (DD) and Zero Noise Extrapolation (ZNE)
applied to dynamic circuit implementations of Hamiltonian simulation.

We study two systems : 1D Heisenberg chain and 1D Transverse Field Ising chain
and perform two sets of experiments on each :
(a) Ground state estimation and (b) Time evolution.

We investigate the effects of DD, ZNE and both DD+ZNE on their effectiveness
in mitigating the errors induced from mid-circuit measurements and feedforward
operations.
